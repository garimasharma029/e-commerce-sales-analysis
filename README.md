E-Commerce Sales Analysis using SQL & Python

## Project Overview

This project analyzes the Olist E-Commerce dataset using MySQL and Python. The data was imported into a MySQL database, queried using SQL, and analyzed with Python libraries to gain business insights.
The project demonstrates SQL querying, data analysis, and data visualization skills.

## Dataset

- Dataset: Olist E-Commerce Public Dataset
- Source: Kaggle

The dataset contains information about:
- Customers
- Orders
- Order Items
- Products
- Sellers
- Payments
- Reviews
- Geolocation

## Technologies Used

## Analysis Performed

Some of the analyses include:
- Total orders by year
- Monthly order trends
- Sales by product category
- Top-selling sellers
- Average order value
- Customer purchase behavior
- Payment analysis
- Correlation between product price and sales
- Revenue ranking using SQL window functions

## How to Run

1. Download the Olist dataset.
2. Create a MySQL database named `ecommerce`.
3. Run `import_data.py` to import all CSV files into MySQL.
4. Run `analysis.py` to execute SQL queries and generate visualizations.
